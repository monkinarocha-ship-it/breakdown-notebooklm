# breakdown-notebooklm
Segundo cérebro sobre a crises mundiais financeiras ciclicas

CONTEXTO E OBJETIVOS

A intenção desse mini projeto e analisar o crises financeiras mundiais e verificar se há alguma tipo de padrão.
As crises financeiras sempre ocorreram ao logos das décadas e por vários fatores, mas será que são cíclicas? ou possui padrões? Por esse motivo criei esse estudo para verificarmos juntos atraves de várias analises com a ajuda da ferramentas de IA

Objetivos principais:
Entender o que causa crises financeiras
Identificar padrões históricos entre diferentes crises
Analisar se existe um comportamento cíclico no mercado
Aprender a utilizar IA de forma mais correta nos estudos
